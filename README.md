# book-store

To install, run:

```
yarn
```

or

```
npm install
```

To start the local dev server (with hot reloading), run:

```
yarn start
```

or

```
npm run start
```

Then navigate to `http:localhost:1324`.

To build, run:

```
yarn build
```

or

```
npm run build
```

NOTE: The app assumes the Book Store API is running on `http://localhost:3000`.
